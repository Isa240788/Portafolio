## 🚀 Stack Tecnológico & Innovaciones

### **Core & Framework**
- **Vue 3 (`<script setup>`)**: Uso de Composition API para una lógica reactiva y limpia.
- **Vite 7**: Tooling de última generación para un desarrollo ultra rápido.

### **Diseño & Estilos**
- **Tailwind CSS 4**: Implementación de la versión más reciente del framework de utilidades.
- **DaisyUI 5**: Componentes UI modernos y accesibles.
- **SASS**: Preprocesamiento de estilos para mayor control.

### **Gráficos & Animaciones**
- **Three.js & OGL**: Integración de elementos 3D para una experiencia inmersiva.
- **AOS (Animate On Scroll)**: Animaciones fluidas al navegar por el sitio.
- **Framer Motion**: Micro-interacciones premium.

## 📁 Estructura del Proyecto (Arquitectura Modular)

```text
src/
├── components/
│   ├── Navbar.vue       # Navegación dinámica con scroll suave
│   ├── Hero.vue         # Sección de impacto con efectos visuales
│   ├── Trayectoria.vue  # Línea de tiempo profesional
│   ├── Expertise.vue    # Grid de habilidades técnicas
│   ├── Proyectos.vue    # Galería dinámica de trabajos realizados
│   ├── Footer.vue       # Redes y contacto final
│   └── ScrollToTop.vue  # Utilidad de navegación UX
├── style.css            # Configuración Tailwind 4 & Globales
├── App.vue              # Orquestador con inicialización de AOS
└── main.js              # Punto de entrada de la aplicación
```

## 🧠 Buenas Prácticas Aplicadas

- ✅ Mobile-First: Diseño 100% responsivo optimizado para todos los dispositivos.
- ✅ Ciclos de Vida: Uso de onMounted y nextTick para asegurar que las animaciones carguen solo cuando el DOM esté listo.
- ✅ Clean Code: Separación clara de responsabilidades entre lógica y presentación.
- ✅ Optimización: Configuración de base en Vite para despliegue perfecto en GitHub Pages.


## 🚀 Instalación

````text
# Clonar repositorio
git clone <url>

# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

#Construir para producción (Build):

npm run build
````

## 📦 Despliegue

Este proyecto está configurado para desplegarse automáticamente en GitHub Pages mediante el comando:

```bash
npm run deploy
```
